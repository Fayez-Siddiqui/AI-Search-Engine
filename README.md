# AI-Search-Engine
Get your firebase_auth_key , firebase_api google_api(gemini), tavily_api_key, langsmith_api_key and databse_url and add it to your keys and .streamlit file.

To run locally
install docker hub
build image
docker build -f "file-name"  -t image-name:tag .
build container
docker run -p 8501:8501 image-name:tag

listen at port 8501
